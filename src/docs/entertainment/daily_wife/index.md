## 主要玩法

每天你都可以通过 **今日老婆** 玩法随机地抽取一位二次元角色作为你的老婆！
你可以自定义各种设置，以决定幻梦每天会给你抽取什么样的角色。

- 你可以指定让老婆每天给你抽一位 {pink}🌸老婆{} 或者 {blue}🔵老公{} ！
- 你可以指定让幻梦每天抽到某游戏/番剧/虚拟主播的概率变高，还可以排除不想要抽到的游戏/番剧/虚拟主播！
- 你可以使用特殊道具来指定你下一次抽到的老婆是谁！
- 通过对每天抽到的角色发表 {green}💖喜欢{} 或 {red}💔不喜欢{} 意见，幻梦会逐渐了解你的喜好，从而提高抽到你喜欢的老婆的概率！

## 换老婆

不喜欢今天的老婆吗？小人类可以点击"换老婆"按钮，消耗相应的额度更换老婆！
- 每日更换老婆没有上限，只要你有非常非常多的额度，就可以做到一直抽老婆一直爽{#8f8f8f}（需注意刷屏问题，避免频繁刷新老婆导致影响到其他人聊天）{}

- 小人类更换了老婆后，可以确定今天的老婆就是她了，也可以继续更换直到抽到喜欢的老婆为止

- 游玩小游戏可以获得额度哦！详情指令: {blue}/小游戏菜单{}

::: info
**关于换老婆的额度消耗**

- 每天 **第 1 次抽老婆免费**。
- 从第 2 次开始，**每多换一次，所需消耗的额度都会比上一次更多**。
- 当日换的次数越多，单次消耗也会越来越高，达到一定次数后会进入"高消耗"区间，请合理规划自己的额度~

具体每一次的额度消耗以幻梦弹出的确认窗口提示为准。
:::

## 偏向和排除

![今日老婆玩法示例图](/img/daily_wife/choose_game_global_example.png){.hm-center-img width="400"}
{gray}(今日老婆玩法示例图){}

你可以设置✅**偏向**来让幻梦每天*尽可能*的抽到你偏向游戏里面的角色，最多可以对**6**个游戏设置偏向。

![偏向设置示例](/img/daily_wife/prefer_game_global_example.png){.hm-center-img width="400"}

你也可以设置 ❌**排除** 来让幻梦每天不抽到你排除游戏里面的角色，最多可以对**6**个游戏排除。

![排除设置示例](/img/daily_wife/disable_game_example.png){.hm-center-img width="400"}

若想取消对某个游戏的偏向或排除设置，对它们再设置一遍即可。

![取消设置示例](/img/daily_wife/cancel_preference_example.png){.hm-center-img width="400"}


偏向的游戏越多，每个游戏提高的权重值越低，如：
1. 小人类偏向了游戏碧蓝档案，那么下一次幻梦抽到碧蓝档案的概率将提升**30%**。
2. 小人类偏向了两个游戏，一个是碧蓝档案、一个是鸣潮，那么下一次幻梦抽到碧蓝档案的概率将提升**15%**、抽到鸣潮的概率将提升**15%**。
3. 小人类偏向了三个游戏，一个是碧蓝档案、一个是鸣潮、还有一个是崩坏三，那么下一次幻梦抽到碧蓝档案的概率将提升**10%**、抽到鸣潮的概率将提升**10%**、抽到崩坏三的概率将提升**10%**。

::: info
这里的概率仅为假设，并非真实概率。
:::

## 常见问题

1. **今日老婆就算我对某个游戏设置了偏向，为什么下一次还是没抽到对应的游戏？**
> 设置偏向不会使每次抽老婆时一定会抽到你设置的偏向游戏，仅提高抽到的概率。
2. **可以指定老婆吗？**
> 可以，小人类需要使用一个道具叫: 定积分券（品质：{orange}传说{}），就可以指定你下一次抽到的老婆啦，指定角色后明天幻梦一定会抽到TA哦。该道具的主要作用就是指定明天的老婆是谁，且不受偏向和排除的影响。
