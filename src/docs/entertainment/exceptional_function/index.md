## 主要玩法
每天都会有一样新的超能力等着你去获取！
你可以获得一项超脱于正常人类的 {blue}超能力{}！
但是也会获得一些 {red}负面效果{} 作为代价！
选择 **✅按下**？还是 **❌不按**？
请小人类认真思考好处与坏处后再做选择~ 你的选择有可能会永久的改变自己，或者其他人的一生。

## 对主题发表你的评论
小人类可以为今天超能力的主题发表你的评论，对其谈谈自己的看法。
幻梦鼓励每个人发表值得讨论，且正确的看法哦！
幻梦支持小人类发表多条评论！但如果要发表第二条评论，必须等待上一条评论审核完毕，才能发表下一条评论。

## 评论加精

![评论被置顶示例图片](/img/exceptional_function/comment_top.png){.hm-center-img width="400"}![评论被加精示例图片](/img/exceptional_function/comment_pinned.png){.hm-center-img width="400"}

如果小人类发表的评论内容较为优质，有可能会被幻梦作者们置顶或加精哦！
被加精的评论前面将会有独特的{red}【置顶】{}或{gold}【精华】{}标志，代表这个评论是优质且经过幻梦作者们二次精选的。
如果小人类的评论被加精，幻梦会额外给予小人类 **20额度** 和 **40点好感度** 作为奖励{gray}（置顶无奖励）{}！
‌
## 评论被驳回和计次

小人类如果发表的评论审核未通过，将会**视情况**增加一次未通过次数。
正常情况下，发表的评论如果因不适合展示而未通过，则不会增加未通过次数，以下行为将会增加未通过次数：
- **与主题无关且无意义的连续数字，字母**
    - 666、1234、aaaaa、114514等。
- **无任何实际意义的评论**
    - 大家好、你们好、第一、沙发等。
- **煽动他人情绪的评论**
    - 类似"傻X才会选按下"、"选不按的都是XXX吧"等。
- **包含真实信息的评论**
    - 类似"王某某别内卷了"、"我实名QQ 3889006601 选择按下"等。
- **与主题无关的梗**
    - 类似"man"、"雪豹闭嘴"等。
