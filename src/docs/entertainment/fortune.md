# 今日运势

## 主要玩法

> 每天小人类都可以抽取今天的运势以及获得对应运势的加成！

**幸运值**：字面意思，代表你今天的幸运值。幸运值和其他大部分玩法挂钩。比如扫雷，幸运值越高，初始雷越少；油漆炸弹，幸运值越高，初始步数越多等等。

**幸运等级**：根据小人类抽到的幸运值分类所属幸运等级。

## 加成池一览

每天抽取运势时，幻梦会从下方的加成池里随机给小人类一条加成，加成的稀有度越高，抽到的概率越低。
