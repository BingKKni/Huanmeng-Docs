# 📅 打卡

### 打卡怎么玩？
每天小人类都可以通过 **打卡** 来获取一定量的 **额度** 和 **好感度** ！
::: info
**额度**是幻梦小游戏里的一种 **虚拟货币**，它可以为小人类解锁加成、升级背包、解锁功能等。
:::

**好感度**是衡量幻梦与小人类之间的亲密程度的重要指标：
- 好感度越多，你的体力上限越高，体力恢复速度就越快。
- 好感度越多，你的打卡时间间隔就越短，这意味着你可以在一天内打更多次卡！
- 好感度越多，你的单次打卡能获得的额度和好感度越多，运势抽到的幸运值越高。

### 一些隐藏小福利
- {green}**首次打卡**{} 第一次成功打卡时，幻梦会额外多给一点点额度作为见面礼~
- {green}**早起的小人类**{} 每天的打卡若进入了当日的前若干名，可以获得一份额外的额度加成。{gray}（具体名次和加成幅度不公开，避免被刷卡）{}

### 🌙 结晶机制
夜深了，幻梦也要休息啦~

如果小人类使用了 **结晶卡** 道具{gray}（可以从一些活动或者保险箱中获得）{}，就能解锁一段时间的"结晶"权益：
- 在 **每天 18:00 至次日 05:00** 这段时间内打卡，获得的额度会进入"结晶"状态。
- 第二天再次打卡时，结晶的额度会转化成 **少量梦晶**（{purple}🟣梦晶{}是一种比额度更稀有的特殊货币）。
- 重复使用结晶卡，可以延长结晶权益的有效期。
