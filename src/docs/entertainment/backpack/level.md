# 🎒 背包等级特权

---

::: warning
**小人类可持有额度数量是有上限的！**

当小人类持有的额度达到了上限之后，将无法再获取额度{gray}（会被吞掉！）{}。所以当持有额度到达上限时，需要及时升级背包，避免后面获取的额度被系统吞掉。
:::

![升级背包按钮示例](/img/backpack/level/level_up_button.png){.hm-center-img width="500"}

{gray}（如果持有的额度到上限了，需要及时点击按钮来升级背包，避免后续获得的额度被吞）{}

## 注释

::: info
- **`*` 幻梦指令**：指绝大部分幻梦的指令，如打卡、运势、翻牌、今日老婆、随机图、扫雷、油漆炸弹等，**不包括** 全局冷却、我的世界查榜指令和其他特定指令。
- 通过升级背包等级获得的指令冷却降低 **最多降低至 2 秒**，如随机图初始指令冷却 5 秒，通过升级背包等级降低冷却不会低于 2 秒。
:::
