# 人设生成
### 创建人设
> 创建一个属于自己的角色设定吧！
你可以通过 **人设生成** 玩法来创造一个 **属于自己的角色设定**，
- 可自定义以下角色属性：
    - 性别（男或女）
    - 年龄（比如16岁，18岁等）
    - 身高（比如155厘米，180厘米等）
    - 体重（比如40公斤，60公斤等）
    - 颜值
    - 智商
    - 情商
    - 体型（比如少女，真男人等）
    - 爱好（比如玩游戏，捣鼓等）
    - 职业（比如学生，律师等）
    - 血型（比如A型血，B型血，O型血，AB型血等）
- 向其他人展示你的OC角色吧~

### 前置要求
由于我们发现部分用户在尝试使用 谐音 等方式绕过机器和人眼审核。出于安全考虑，我们不得不增加了设定人设的前置条件，只有小人类满足全部要求了才可以开始创建人设。{#8f8f8f}（已创建但未达到要求的不受影响）{}
要开始创建人设，必须满足以下要求：
1. 背包等级到达12级及以上。
2. 好感度等级达到C4级及以上。
3. 今日超能力玩法中，拥有对今天主题发表评论的资格。{#8f8f8f}（因某原因导致小人类被取消发表评论资格的，将无法满足该要求）{}
4. 剧情模式玩法中，拥有对剧情发表评论的资格。{#8f8f8f}（因某原因导致小人类被取消发表评论资格的，将无法满足该要求）{}

### 修改已创建的角色设定
> 小人类可以随时修改已经审核通过且创建成功的角色设定。

单次修改角色设定需消耗 `10~1000` 额度。如果你是第一次编辑角色，则只用消耗 `10` 额度，后续编辑的次数越多，消耗的额度越多。

**编辑分为两种模式:**
1. 全部。可修改全部人设设定。（昵称，性别，身高体重，颜值情商智商，血型等）（审核速度较慢，因为要重新复查小人类创建的角色设定）
2. 部分。除修改颜值、情商、智商这三个数值设定外，其余的都可以修改。
::: warning 注意
如果小人类在修改人设时选择 "**全部**" ，则提交后需要再次经过审核才可继续展示，请在编辑之前想清楚是否真的需要编辑人设设定。
:::
### 关于审核
- 出于安全考虑，小人类在创建角色设定后，首先会经过一次机器初审，若机器初审未通过{#8f8f8f}（审查出了违规内容）{}，就需要复查前面的设定是否出现了违规内容{#8f8f8f}（如黄赌毒）{}，是否包含违反幻梦使用协议的内容。
- 机器初审通过后，数据会统一提交给幻梦的审核同学再次进行人工复核。需要说明的是：复核的目的只是检查有无违规元素{#8f8f8f}（涉嫌黄赌毒或其他不适宜展示的内容等）{}，如果没有审查出违规和不适宜元素，则审核通过，视为创建人设成功；否则人设申请将被拒绝，小人类在提交人设审核后，如果人设被拒绝了，可以根据未通过的原因修改人设后再次提交审核哦。
- 小人类应当为自己设定的人设负责。如果审核同学发现小人类尝试用 *谐音* 等方式绕过机器审核、人工审核的，审核同学有权将你直接拉入幻梦黑名单，一旦被拉入黑名单，将无法再使用幻梦的任何功能哦。

### 常见问题
1. 审核的时间一般是多久？
> 正常情况下，审核人设大概耗时3至10个工作日，如果审核同学平时较忙，审核人设所需时间还会相应延长，最长不会超过10个工作日。
2. 为什么人设审核给我驳回了？
> 驳回人设时会附带相应的驳回理由，请大大根据驳回理由对设定的人设进行改正，方可继续提交审核。
> 出于安全原因，在预览设定的人设时，幻梦不会显示出包含用户自主输入的内容（昵称、职业等，这些内容会强制打码），如果你因为以上设定（昵称、职业等）被驳回了，看不到自己设定了什么，请查看你与幻梦的聊天记录，或者干脆直接重新设定一个即可。
3. 对已通过的角色编辑部分设定后，还要再次提交审核吗？
> 是的。但如果小人类只编辑了颜值、情商、智商这三个数值设定，则在编辑完提交后无需经过审核即可生效新设定。如果编辑了这三个数值设定以外还编辑了其他数据，则在提交后需要经过审核才能生效。
