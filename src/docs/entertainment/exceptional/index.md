## 主要玩法
每天都会有一样新的超能力等着你去获取！
你可以获得一项超脱于正常人类的 {blue}超能力{}！
但是也会获得一些 {red}负面效果{} 作为代价！
选择 **✅按下**？还是 **❌不按**？

请小人类认真思考好处与坏处后再做选择~ 你的选择有可能会永久的改变自己，或者其他人的一生。
::: info ㅤ
每天只能选择一次。若不小心按错了可以在10秒内点击"撤回选择"蓝色字体来撤回你的选择，每天最多撤回一次选择。
:::
## 对主题发表你的评论
小人类可以为今天超能力的主题发表你的评论，对其谈谈自己的看法。
幻梦鼓励每个人发表值得讨论，且正确的看法哦！
幻梦支持小人类发表多条评论！但如果要发表第二条评论，必须等待上一条评论审核完毕，才能发表下一条评论。
## 评论加精
![评论](/img/exceptional/exceptional2.png){.hm-center-img width="400"}

如果小人类发表的评论内容较为优质，有可能会被幻梦作者们加精哦！
被加精的评论前面将会有独特的 🌟 标志，代表这个评论是优质且经过幻梦作者们二次精选的。
如果小人类的评论被加精，幻梦会额外给予小人类 **20额度** 和 **40点好感度** 作为奖励！
‌
## 评论被驳回和计次
小人类如果发表的评论被管理员驳回，则可能会增加一次`计次` 。
**如果小人类发表的评论是和今日超能力主题有关系的，但评论不适合给其他小人类展示，那么只会驳回不会计次，驳回后小人类还可以继续发表评论。**
以下评论将会在审核人员驳回的同时计次次数：
- **与主题无关且无意义的连续数字，字母**
    - 666、1234、aaaaa、114514等。
- **无任何实际意义的评论**
    - 大家好、你们好、第一、沙发等。
- **煽动他人情绪的评论**
    - 类似"傻X才会选按下"、"选不按的都是XXX吧"等。
- **包含真实信息的评论**
    - 类似"王某某别内卷了"、"我实名QQ 3889006601 选择按下"等。
- **与主题无关的梗**
    - 类似"man"、"雪豹闭嘴"等。
‌![驳回计数](/img/exceptional/exceptional3.png){.hm-center-img width="400"}
该数值如果超过4，则小人类将无法再发表评论，只能前往幻梦官方群说明情况以解除限制。
