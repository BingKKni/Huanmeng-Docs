# 体力

## 主要说明
体力可以用于游玩幻梦的小游戏，如速算、油漆炸弹等~
体力每个人默认上限为 1200 点。
体力恢复速度与幻梦的[好感度](sence?from=stamina)有关，[好感度](sence?from=stamina)等级越高，代表幻梦越加亲近你，体力回复速度越快。
