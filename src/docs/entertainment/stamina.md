# 体力

## 主要说明
体力可以用于游玩幻梦的小游戏，如速算、油漆炸弹等~
体力每个人默认上限为 1200 点。
体力恢复速度与幻梦的[好感度](sence?from=stamina)有关，[好感度](sence?from=stamina)等级越高，代表幻梦越加亲近你，体力回复速度越快。

当体力值到达上述对应等级的体力最高上限后，将不再恢复。
小人类可以使用部分道具来加快体力恢复速度，或提高上限~

::: info 关于临时加成

上方表格里的"恢复速度"和"体力上限"是 **基础值**。

当小人类使用了 {blue}🟦体力恢复剂、运动饮料、肾上腺素、兴奋剂{} 等保险箱道具时，会获得 **临时加成 BUFF**：
- 临时加成会在表格基础值上 **额外叠加** 一段时间。
- 加成时长结束后，自动回到基础值。

具体每个道具的加成数值和时长，可以在 {blue}/保险箱{} 内查看对应道具的描述。
:::

::: info 每日恢复一次体力至上限

当小人类的 [背包等级](backpack/level) 达到一定等级，并且 [好感度](sence?from=stamina) 等级也达到对应等级时，可以解锁此福利：每天可以手动恢复一次体力到当前的上限~

具体的解锁条件可以在 [背包等级特权](backpack/level) 页面查看。
:::
