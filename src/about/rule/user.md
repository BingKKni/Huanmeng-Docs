# 幻梦Bot用户服务准则

更新日期：2026年4月20日
生效日期：2026年4月21日

幻梦Bot（以下简称“幻梦”）是一个运行在QQ软件上的娱乐型互动机器人，幻梦提供游戏、玩法、互动和各类自定义功能。为了让每个人都能在一个良好的环境里使用幻梦，我们制定了以下准则。

使用幻梦（包括邀请、调用、游玩、上传内容、其他互动等任何形式的使用），即视为您已阅读并同意遵守本准则。如不同意，请立即停止使用幻梦服务。
部署的含义：邀请、添加、绑定、拉入等将幻梦引入一个群聊或频道或您个人私信列表的操作。

---

## 一、适用范围

本篇准则覆盖幻梦提供的全部服务。无论您是普通用户还是将幻梦部署到群聊或频道的管理员，本准则均对您适用。

- 本准则适用于幻梦在QQ私聊、QQ群聊、QQ频道及其他支持场景中的全部服务，包括指令调用、游戏玩法、互动回复、虚拟货币与道具、好感度、额度、称号等虚拟数据，以及人设生成、头像上传、评论区、公开展示等自定义功能，还有相关帮助文档、活动、公告、反馈与其他管理功能。
- 本准则适用于所有使用幻梦的用户，包括邀请幻梦进入群聊或频道的管理员，主动或被动添加幻梦为QQ好友、以及通过他人QQ账号、QQ群聊、QQ频道间接使用幻梦的用户。
- 使用幻梦的同时，您也需遵守QQ平台的用户协议、社区规则与其他管理规范，以及适用的法律法规。

## 二、基本行为准则

幻梦所在的群聊和频道，是许多人共同使用的空间。我们希望您的群聊或频道能保持轻松、有趣且互相尊重的氛围，让所有人都能安心游玩幻梦。任何破坏这种氛围的行为，都可能影响到其他用户的体验。

以下行为是禁止的：

1. 辱骂、骚扰、威胁、攻击、歧视或诽谤幻梦的；
2. 组织或参与针对幻梦的恶意攻击；
3. 传播关于幻梦的虚假信息、恶意谣言或可能引发公共误解的内容的；
4. 利用幻梦进行刷屏骚扰、特殊引流、诈骗、赌博或其他恶意营销的；
5. 冒充幻梦官方、幻梦管理员、幻梦开发者或其他用户的；
6. 以任何方式干扰幻梦的正常运行或使用秩序的。

## 三、邀请与管理规范

为了保障幻梦的每位用户的使用体验，以及确保幻梦的有限服务资源不被滥用，我们对邀请和管理行为作出以下要求。

1. 邀请幻梦进入群聊或频道前，您应确保已取得群主、管理员或频道管理方的授权或同意。
2. 使用幻梦时，不得规避平台方对机器人、群聊、频道或账号的管理规则。
3. 群聊或频道管理者应对幻梦的使用环境负责。若群聊或频道长期存在恶意刷屏、违规内容、攻击幻梦或滥用功能等情况，幻梦管理员和开发者有权限制或停止对相关群聊、频道的服务。
4. 通过多个账号、群聊或频道规避处罚、限制或风控的行为，将被视为加重违规处理。

## 四、内容安全规范

幻梦的使用场景覆盖不同年龄段、不同背景的用户。我们需要确保通过幻梦输出的内容不会对任何人造成伤害，也不会让幻梦成为传播有害内容的工具。

您通过幻梦发送、上传、展示或传播的任何内容，均须符合法律法规、公序良俗及平台规则。

### 4.1 禁止内容

禁止通过幻梦发送、上传、展示或传播以下内容：

1. 违法违规内容；
2. 色情、低俗、露骨或带有性暗示的内容；
3. 暴力、血腥、恐怖、极端或引人强烈不适的内容；
4. 涉及诈骗、赌博、毒品、违禁品或其他违法活动的内容；
5. 恶意程序、钓鱼链接、病毒木马或其他危害网络安全的内容；
6. 泄露、收集或传播他人隐私及个人信息的内容；
7. 侵犯他人著作权、肖像权、名誉权、隐私权或其他合法权益的内容；
8. 可能引发严重对立、攻击或现实风险的敏感内容。

### 4.2 禁止恶意触发

为了确保幻梦的输出安全，幻梦设立了一定的安全限制机制。这些机制的存在仅是为了保护幻梦的输出安全，而不是视为挑战的对象。故意绕过或测试这些限制，本身就是一种违规行为。

不得通过任何方式恶意触发幻梦输出违规内容，包括：

1. 利用谐音，语义暗示、变体表达等方式绕过安全限制；
2. 故意引导幻梦输出违法、色情、暴力、仇恨、诈骗等内容；
3. 反复试探幻梦的风控机制或安全边界；
4. 将违规意图伪装成测试、玩笑、虚构设定或所谓的技术讨论。

一经发现，幻梦管理员或开发者有权限制或永久停止向相关用户、涉及群聊或频道提供服务。

### 4.3 未成年人保护

幻梦的用户群体中可能包含未成年人。保护未成年人不受有害内容的影响，是所有用户共同的责任，也是我们最基本的要求之一。

1. 禁止通过幻梦发布、传播任何可能危害未成年人身心健康的内容，包括涉及未成年人的色情低俗内容、暴力欺凌内容、不良诱导内容，以及披露未成年人隐私的内容；
2. 禁止利用幻梦对未成年人实施骚扰、欺骗、引诱或其他可能造成伤害的行为；
3. 如您为未成年人，建议在监护人了解并同意的前提下使用幻梦服务；
4. 若发现涉及未成年人的违规内容或行为，欢迎通过官方渠道及时反馈，幻梦管理员或开发者将予以优先处理。

## 五、自定义功能规范

人设生成、头像上传、背景设置、评论区、及其他公开展示等包含自定义的功能，是为了让每个人都能在幻梦里表达自己、丰富玩法体验而设计的。这些功能会直接呈现给其他用户看到，因此内容上的边界同样重要。

以下行为是禁止的：

1. 设置包含违规内容、色情低俗、暴力血腥、侮辱歧视、恶意引战或带有敏感暗示的名称、背景、简介或人设信息；
2. 上传违规、侵权、低俗、血腥或恶意引流的图片作为头像、背景或其他展示素材；
3. 在评论区或公开展示区发布广告、辱骂、涉黄涉政、诈骗链接或其他违规内容；
4. 冒用他人身份、盗用他人图片，或上传您无权使用的内容；
5. 利用自定义内容误导他人，使其误认为您与幻梦官方存在合作、授权或管理关系。

您应确保所上传、填写、发布的内容拥有合法权利或必要授权。因用户内容产生的纠纷，由内容发布者自行承担责任。

## 六、游戏公平与禁止作弊

幻梦的达尔文游戏系统建立在所有人平等参与的基础上。作弊行为不仅对认真游玩的用户不公平，也会破坏整个玩法生态。为了维护这份公平，我们需要所有人通过正常方式参与玩法、获取奖励和积累数据。

以下行为是禁止的：

1. 使用外挂、脚本、按键精灵、点击器、自动化程序或其他手段自动游玩、自动签到（打卡）、自动刷取奖励；
2. 利用系统漏洞、逻辑缺陷、数据异常或结算错误刷取梦晶、道具、好感度、额度、称号或排行榜数据；
3. 通过异常频率、异常路径或异常账号关系获取游戏奖励；
4. 传播、出售或分享作弊方法、漏洞利用方案或自动化脚本；
5. 发现漏洞后不反馈，反而继续利用或引导他人使用；
6. 通过小号、代刷、互刷、恶意转移或异常交易破坏玩法平衡。

发现上述行为，幻梦管理员或开发者有权采取数据回档、收益清除、道具回收、排行榜移除、功能限制、账号封禁、群聊封禁或频道封禁等措施。

如果您在游玩中发现了漏洞或异常，欢迎通过官方渠道反馈。这对维护所有人的游戏体验很有帮助，我们也非常感激。

## 七、合理调用与反滥用规范

幻梦的服务资源是有限的，异常的调用行为可能导致其他用户响应变慢，甚至影响整体稳定性。为了让每个人都能正常使用，我们需要所有用户合理调用幻梦，不占用超出正常使用范围的资源。

以下行为是禁止的：

1. 短时间内过于频繁、重复地调用幻梦指令；
2. 连续请求间隔小于1秒，或以明显异常的频率发送请求；
3. 使用自动化工具或脚本批量调用幻梦；
4. 大量尝试不存在的指令、参数，以探测幻梦的插件、框架或系统结构；
5. 对幻梦进行未经授权的压力测试、扫描、爬取或逆向工程；
6. 利用漏洞、并发机制或平台延迟等手段绕过调用限制；
7. 发送大量无意义或干扰性内容；
8. 滥用反馈渠道发送垃圾信息或进行恶意投诉。

异常使用可能触发自动风控，幻梦有权对相关账号采取临时限制、长期限制或永久封禁。

幻梦的稳定运行依赖于底层系统的安全与完整性。任何对系统的未授权访问或破坏行为，都可能影响到所有用户的数据和使用体验，这是我们绝对不允许的。

禁止对幻梦及其相关系统实施以下行为：

1. 未经授权访问、探测、扫描或攻击幻梦服务器、接口、数据库或管理系统；
2. 逆向、反编译、破解、篡改、注入或劫持幻梦相关程序或接口；
3. 使用爬虫、批量请求或抓包等方式系统性收集幻梦数据、用户数据或玩法资料；
4. 绕过权限限制、风控限制、封禁限制或平台管理规则；
5. 获取、传播或交易幻梦源代码、密钥、配置文件、数据库或其他未公开资料；
6. 干扰幻梦的正常运行，或影响其他用户的正常使用。

违反本条的，幻梦管理员或开发者有权立即停止向相关用户、群聊或频道提供服务，并保留进一步处理的权利。

1. 本准则可能根据服务变化、功能调整、平台规则或管理需要不时更新。
2. 准则更新后，将通过[幻梦菜单和攻略帮助](/docs/)或其他官方渠道进行公示。
3. 更新后的准则自公示或标明的生效日期起适用。用户在准则更新后继续使用幻梦服务的，视为已同意遵守更新后的内容。
4. 若不同意更新后的准则，请停止使用幻梦服务。
